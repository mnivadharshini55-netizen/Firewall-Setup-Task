# Firewall-Setup-Task
# 🔐 Cyber Security Internship - Task 4

## 📌 Title

Setup and Use a Firewall on Windows

---

## 🎯 Objective

To configure and test basic firewall rules to allow or block network traffic using Windows Firewall.

---

## 🛠️ Tools Used

* Windows Defender Firewall

---

## ⚙️ Steps Performed

### 1. Open Firewall Settings

* Opened Windows Security
* Clicked on Firewall & Network Protection
* Verified firewall is enabled

### 2. View Existing Rules

* Opened Advanced Settings
* Checked Inbound Rules

### 3. Create Firewall Rule

* Selected New Rule → Port
* Chose TCP
* Entered port number 23
* Selected "Block the connection"
* Applied to all profiles
* Named rule as **Block Port 23**

### 4. Test the Rule

* Attempted to access port 23
* Connection was successfully blocked

---

## ✅ Result

Inbound traffic on port 23 was successfully blocked using Windows Firewall.
